# TeleX TV Store

**TeleX** is a web-based e-commerce application specialized in selling televisions. The project provides functionality for searching, filtering, comparing products, user management, and displaying promotions.

---

##  Main Features
- Pages: **Home**, **Register**, **About**, **Products**, **Contact**
- User registration and authentication
- Product listing with detailed information (**price**, **screen size**, **warranty**, **quality**)
- Automatic price calculation with discount
- **Filtering** by:
  - **Screen size**
  - **Price**
  - **Warranty**
  - **Category**
  - **Quality**
- Product comparison
- Business hours and informational sections (announcements, gallery, FAQ)
- Store location and contact details
- Active discounts and promotions

---

##  Similar Websites

1. [emag.ro](https://www.emag.ro/search/televizoare/televizoare/c?ref=search_category_1)  
**Pros:** Offers advanced filtering and short descriptions for each product including specs, rating, and price.  
**Cons:** The layout is cluttered and text size is small.

2. [altex.ro](https://altex.ro/televizoare/cpl/)  
**Pros:** Clean and simple design, easy-to-use filters.  
**Cons:** Fewer filtering options compared to competitors. Important links are hard to reach due to too many products on one page.

3. [mediagalaxy.ro](https://mediagalaxy.ro/televizoare/cpl/)  
Very similar to Altex, with the same strengths and weaknesses.

4. [flanco.ro](https://www.flanco.ro/tv-electronice-foto/televizoare.html)  
**Pros:** Useful links are easily accessible. Products come with concise info like specs, rating, and energy class.  
**Cons:** Filter section is compact with small text, which makes it feel crowded.

5. [ideall.ro](https://www.ideall.ro/magazin-televizoare)  
**Pros:** Product details are clear, navigation is intuitive.  
**Cons:** Limited filtering options; footer links are all squeezed into one section.
